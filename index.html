Model Scoring

DeepTzu automatically evaluates the quality of your model in a comprehensive and systematic way. For instance, 1) how vulnerable of your model under state-of-the-art adversarial attacks using a set of attack metrics; 2) how robust your model is using a set of robustness metrics; 3) how well your model is tested using a set of testing metrics.

Model Enhancement

DeepTzu automatically enhances the quality of your model using a pipeline of state-of-the-art model enhancement techniques including data augmentation, model retraining, adversarial training, robust training, etc.

Real-Time Safety Assurance

DeepTzu provides an enhanced model together with modules which can protect your model with real-time safety assurance by state-of-the-art techniques to detect and filter adversarial samples at runtime including mutation testing, invariant check, LID, etc.
